---
description: Aggressively clean up code comments in a zone (auto-selects one if no path is given)
argument-hint: "[<path>]"
---

Invoke the doc-cleanup skill in **zone** mode. Arguments: $ARGUMENTS

If `$ARGUMENTS` is empty, auto-select a zone (inventory + autonomous selection, validated with the user). If `$ARGUMENTS` is a path (file or directory), clean that zone directly. Read `references/doctrine.md` first, then follow "Mode : zone" (`references/mode-zone.md`) in the skill.
